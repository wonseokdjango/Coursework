Theory Of Computation HW 1
Baker Bird 2d Pattern Matching

2017-21586 Wonseok Lee

- Environment
  OS       : Ubuntu 14.04.5 LTS
  Language : C++
  Compiler : g++ 4.8.4

- Quick start
  You can enter './run.sh'
  This shell execute hw1 and checker using 10 input files in bb_input/
  Output files of Baker Bird algorithm are stored at bb_output/
  Output files of checker program are stored at cc_output/

- How to Compile
    hw1 compile     : g++ -std=c++11 -o hw1 hw1.cpp
    checker compile : g++ -std=c++11 -o checker checker.cpp

- How to Run
    hw1 run         : ./hw1 bb_in.txt bb_out.txt
    checker run     : ./checker bb_in.txt bb_out.txt cc_out.txt
